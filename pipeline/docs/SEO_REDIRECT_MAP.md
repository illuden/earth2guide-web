## 4. 다른 처리 (미들웨어/로직)

- locale 없는 URL (예: `/news/abc`) → next-intl 미들웨어가 자동으로 `/ko/news/abc` 또는 Accept-Language 따라 분기
- `/wiki?category=essence|jewel|raid|general` (옛 query param) → next.config has 조건으로 308 redirect → `/ko/wiki/overview`
