# 市场

市场是玩家可以交易其资产的地方。

要访问市场，请点击屏幕顶部导航栏中的 [Marketplace](https://app.earth2.io/#marketplace) 按钮。

![image](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/wiki/2dde849c_marketplace-1.png)

在此页面上，您可以使用**筛选器**搜索特定报价。

**市场价值历史**将带您进入[国家统计](https://app.earth2.io/#statistics)页面，该页面显示有关每个国家价值的更多信息。

请注意，我们会不时更新网站，重新设计并添加新功能。

![image](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/wiki/66694c47_marketplace-2.png)

该页面将显示多个结果。所有待售资产均由 Earth 2 的玩家提交。

- Tiles：资产的 tile 大小。

- Tier：资产的 tier。

- Class：资产的 Class 编号。

- Country：资产所在的国家/地区。

- Essence Promised：附加到资产的承诺 Essence 数量。

- New Land Value：在该国家/地区新购买的相同大小资产的价值。

- Name/Advertised Price：资产的名称和销售玩家设定的价格。

- % vs New Land Value：新土地价值和广告价格之间的价格差异。

- Actions：查看或购买市场中列出的资产。

![image](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/wiki/c0a45c20_marketplace-3.png)

**View** 按钮将带您进入资产的详细信息页面，显示有关所选土地的有用信息。

**Buy** 按钮将带您进入网格视图，您可以在继续购买之前查看位置、大小、形状和价格。

您可以使用**Account Balance**付款，款项将从您的账户余额（白色）中扣除。您可以通过访问您的个人资料设置页面向您的账户余额添加更多资金。或者，您可以使用当前支持的**Credit Card**付款。

请注意，我们正在积极努力添加更多付款方式，因此某些服务可能会不时不可用。如果您在使用信用卡购买时遇到任何问题，请联系您的服务提供商以寻求更多信息。Earth 2 采取措施减少欺诈交易，但不幸的是，这有时会导致合法的购买者在购买时遇到问题。

要了解更多关于如何在市场上列出您自己的资产以及竞标或购买土地的信息，请访问之前的操作指南。
