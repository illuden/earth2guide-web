# 自定义推荐码、解锁国家及更多

![news](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/e2_lockedcountries.png)

2022年2月10日 – 悉尼。

亲爱的Earth 2玩家们，我们希望一切都好，并且很高兴在两天内为您带来第二次更新。

T1地产所有者的自定义推荐码

对于我们的EDC Tier 1地产所有者，承诺的自定义推荐码福利已经实施并已上线！回顾一下，推荐码福利包括：

- 从Earth 2购买新的Tile时，代码所有者可获得7.5%的信用额度，代码使用者可获得7.5%的折扣

- 代码自定义功能，最多允许15个字符

这是EDC（https://earth2.io/edc）承诺的一部分，我们很高兴我们的T1地产所有者可以独特地使用他们自定义的推荐码，为使用它们的任何人提供和获得更大的折扣和信用额度！

即将解锁的锁定国家

除了推荐码更新之外，Earth 2 正式宣布计划立即开始解锁“锁定”国家。对于E2社区的许多成员来说，这将是一个令人兴奋的消息，特别是那些一直在等待在这些锁定国家购买土地的人。

所有锁定的国家将以Tier 2 Tile的形式发布，并且可以以每个Tile E$0.10的标准基本价格购买。由于我们的开发团队在相关问题区域内的位置数据方面所做的改进，发布这些锁定的国家/地区已成为可能。这些区域内的位置数据现在得到了准确的反映，我们很高兴最终能够让Earth 2玩家在以前锁定的国家/地区获得所有权。

应该适当注意的是，我们的团队并没有忘记在锁定国家拥有T1土地的玩家，出于善意，我们正在研究为这些玩家及其受影响的T1地产提供补偿的方法。

此外，我们已决定错开所有38个锁定国家的发布时间。不会有关于何时发布某个国家的官方公告，并且发布将在不同的时区进行，以确保来自世界各地的玩家都有公平的机会。

这些发布的国家/地区最大的变化是，在某个国家/地区正式发布后的设定时间内，用户每天可以购买的 Tile 数量将受到限制。这不仅可以防止少数玩家以最优惠的价格囤积早期购买的 Tile，而且还将增加该国家/地区所有权的多样性，包括未来投票活动的多样性，对此我们感到非常兴奋。

鉴于我们最近发布的关于 Resources 的公告，我们了解到当这些新国家/地区上线时，许多玩家将会关注 EcoSim 驱动的购买，因此我们最好的建议是保持警惕，并通过倾听社区的声音和监控社交媒体上的讨论来保持警惕。

其他更新

关于 Essence，我们还添加了 E-ther 热图查看选项。玩家现在可以在地图上看到 Mentar 在世界各地检测到的 E-ther 位置！此功能是未来可能发生的更大变化的第一步，其中 E-ther 检测率的集中度可能会对各种游戏机制产生共同影响。它还继续展示了我们捕获数据并在全球地图上显示相关指标的能力，随着时间的推移，这将随着 Earth 2 中其他数据源的加入而得到扩展。

其他更新包括：

- 最新的 mapbox 地图版本集成

- 财产 Portfolio 查看更新

- P2P 赠送功能更新

- Holobuilding 编辑器的持续改进（更多功能即将推出）

有关所有 E2 更新的更多详细信息，请访问 https://earth2.io/updates

关于 Earth 2

Earth 2® 是第二个地球的未来概念；一个元宇宙，介于虚拟和物理现实之间，其中真实世界的地理位置对应于用户生成的数字虚拟环境。这些环境可以被拥有、购买、出售，并且在不久的将来可以进行深度定制。
