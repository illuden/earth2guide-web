# 宝石

![news](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/jewels_previous-1-1.jpg)

亲爱的 Earth 2 ® 用户：

希望一切都好。几个小时前，我们悄悄地推出了用于整个 Earth 2 ® 应用程序界面的新 React 前端框架。虽然这些变化在视觉上不太容易识别，但此次更新意义重大，涉及数月的工作。此次更新的益处非常重要，因为它将帮助我们避免累积过多的技术债务，而这正是我们在使用 Riot 时所面临的问题。基本上，这意味着我们可以更快地推出和调整功能，这些功能更安全且需要的维护更少。

React 的更新幅度很大，我们恳请您耐心等待，因为我们需要识别和稳定可能出现的任何问题。我们的内部 QA 团队在过去一周一直专注于此次发布的最终测试，但由于我们的预生产环境和生产环境之间存在一些未知的变化，因此我们一直在积极努力解决许多问题。Earth 2 ® 社区管理团队也在监控社交媒体，以便将错误报告给我们的 DEV 团队，因此通过此类渠道报告问题有助于我们更快地定位和修复问题——谢谢。

此次 React 更新非常重要，需要在任何重要的 Phase 2 功能发布之前上线进行监控和稳定。我们还选择在 React 更新的同时推出一个名为 Jewels（宝石）的初始 Phase 2 功能。这是为了启动 Phase 2。如果一切按计划进行，我们将考虑在 2-3 周内发布以下计划的 Phase 2 功能。

宝石

Jewels（宝石）可能会随机出现在用户拥有的土地上，并且在消失之前只有有限的时间可以被认领。用户可以使用 Jewels（宝石）来调整和优化其土地、生产等的持续产出和功能，未来还将有更多用途。

最初生成的 Jewels（宝石）将指示其在整个 Phase 2 中的特殊能力，但是，Jewels（宝石）很快就可以被摧毁，从而产生碎片，然后可以对这些碎片进行制作、合并和组合，以发现具有额外好处的新的且可能稀有的 Jewels（宝石）——因此请尽可能多地保留您可以认领的 Jewels（宝石）！我们还将推出一个 Bazaar（集市），允许用户交易他们的 Jewels（宝石）。

创造力——打开大门

创造力对于 Earth 2 ® 来说非常重要，虽然我们当然专注于创建强大的基础，但我们不能如此大胆地声称我们应该是参与 Earth 2 ® 将成为的唯一创意人员。

我们的愿景一直是将 Earth 2 ® 打造为一个平台，除了我们发布的基础和游戏之外，用户生成的内容、概念和想法也可以随时可用。

因此，我们很高兴地宣布 Earth 2 ® 即将开始创建官方 API 套件，我们希望随后推出 SDK。这将使我们能够欢迎来自世界各地创作者的创意、想法、游戏等。只有在商业、技术和支持方面实现更好的稳定性后，才能实现 API 套件的创建。这是我们在过去 6 个月中努力构建的——我们觉得我们非常接近实现目标。

我们预计许多有趣的想法和机会将向我们的 Earth 2 ® 用户开放，并计划在 2021 年底之前准备好最初的 Earth 2 ® API 套件。我们热切欢迎社区参与第一个版本的 API 请求。

想法和可能性是无限的，我们迫不及待地想看看人们创造什么并将其插入 Earth 2 ®！

最后，我们即将完成一项收购，这不仅将有助于满足 Phase 3 的技术要求和进展，还将使我们能够创建第一个与 Earth 2 ® 平台链接的 PvP 游戏的官方示例！

这款特殊的游戏可能还需要几个月的时间才能推出，但我们非常高兴能在发布之前进行开发。我们计划将其展示为一个开创性项目，这将帮助我们实现并为最初的 API 和 SDK 计划奠定基础。

下一步

目前，我们专注于确保新的 React 框架有效运行，并修复升级带来的任何问题。未来的开发将主要集中在 Phase 2 功能的发布上。Holobuildings 即将到来，Essence 也是……也许还有 E-ther。
