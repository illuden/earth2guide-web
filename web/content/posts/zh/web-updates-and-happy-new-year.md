# 网页更新及新年祝福！

![news](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/hny-e2.jpg)

亲爱的Earth 2玩家们，

希望一切都好，新年快乐！

我们参加新年派对稍微晚了，但我们的团队在假期里一直在努力进行一些重要的修复和更新。为了延续我们在2021年底的发展轨迹，我们计划在2022年全年尽可能频繁地向我们的玩家和社区提供持续的更新、文章和计划。除了几小时前部署的网页更新之外，您还可以期待下一层级的 Jewel、一个完全改进的竞标系统以及一个简短但内容丰富的开发者视频博客在未来几周内上线。

最近的网页更新包括对头部用户界面的全面改进，该界面具有更加用户友好的地图查看模式，以及重新定位的搜索栏，现在方便地位于其旁边，以将所有与地图相关的功能集中在一个位置。

“操作指南”、“支持”和“新闻”按钮也已添加到主标题中，以便玩家登录后可以更轻松地访问，所有这些都使得使用移动设备的玩家可以访问更多功能并轻松访问。

其他一些已上线的重要的更新包括修复了 Holobuilding 编辑器，取消了对可编辑蓝图的限制，始终保持“克隆”和“删除”操作按钮可用，并保持 Holobuilding 在放置于不同地块上时与创建它们的地块保持比例。还优化了编辑器上的 Tile 生成。这些更改为我们的 E2 创作者提供了更愉快的创作体验。

像这样的小改动，尤其是网页 UI 更新，是我们发布 Earth 2 启动器第一个版本的长期目标的一部分，这使我们不仅可以通过我们的网站提前发布启动器的预期 UI 的一部分，还可以测试该 UI 在实际用户中的运行情况。

与某些人可能说的相反，构建元宇宙可以而且应该通过较小的步骤来实现，从在元宇宙中具有长期根本相关性的功能开始。

玩家的创造力、交易、游戏、机会、所有权、合作、奖励和投票是一个开始，也是必须的，我们将继续努力设置这些较小的基础，同时致力于更大的、长期的第三阶段愿景。

感谢您成为这段旅程的一部分，并祝您，Earth 2 最重要的部分，2022 年快乐！

为了新的一年开始，我们更加接近我们的目标！

https://app.earth2.io/#thegrid – 查看新的页眉UI更新

https://earth2.io/updates – 随时了解此版本和其他版本的详细信息。这是第 48 个“官方”更新。

关于 Earth 2

Earth 2® 是一个关于第二个地球的未来概念；一个元宇宙，介于虚拟和物理现实之间，其中真实世界的地理位置对应于用户生成的数字虚拟环境。这些环境可以被拥有、购买、出售，并在不久的将来进行深度定制。
