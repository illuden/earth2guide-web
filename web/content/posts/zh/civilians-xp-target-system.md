# 平民经验值系统 + Raid指挥官目标选择系统

![news](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/05_12_web_civilians_xp_raid.jpg)

2023年5月17日 – 澳大利亚悉尼 – 在平民部署到我们的第二阶段预生产服务器之前，我们想向Earth 2社区更新关于新的平民经验值和Raid指挥官自动目标选择系统如何运作的信息。

我们还想指出，红色能量维持、平民经验值和扩展的Raid指挥官自动目标选择系统是对最初平民概念的新扩展，这导致了之前计划发布的初始平民系统的延迟。

“这些新系统是深入而复杂的，我想花点时间感谢我们的BE开发人员、FE开发人员、游戏设计师、QA、UI/UX团队以及其他人员，他们设法设计了机制、设计了界面、实现了后端、编码并连接了前端，并在创纪录的时间内运行了初步的质量保证检查，考虑到这些新的额外扩展有多复杂，以及它们需要如何合并到原始概念中”，Earth 2创始人Shane Isaac说。

“新引入的游戏机制有助于形成更强大的游戏循环，奖励那些通过技能更好、功能更强大的平民取得进展的玩家，允许Raid指挥官瞄准多个地块，引入了维持概念的早期版本，该概念与直接的玩家活动相关联，提供了新的战略玩法，并且非常重要的是，它允许我们创建、设计和开发系统，我们可以对其进行测试，以便在EcoSim的其他部分中计划采用。”

平民经验值系统

虽然平民将帮助玩家管理他们在Earth 2中的地块和任务，但提高平民的能力需要的不仅仅是E-ther和红色能量来合成和维持他们。这就是新的平民经验值系统将派上用场的地方。

平民经验值系统将允许玩家提升其平民的等级，通过实践经验提高生产力并获得新技能。第一批平民将协助玩家执行他们的Raid活动，当在该地块上分配E-ther时，他们将从成功的Raid中获得经验值。最初，每分配0.01 E-ther将等于1经验值。

充能、Raid和分配需要平民团队的共同努力，因此，经验值将在玩家分配给每个特定地块的Raid系统的所有平民之间平均分配。如果玩家在一个地块上有3个平民，并且该地块分配了3 E-ther，那么将获得300经验值，每个平民获得100经验值。

玩家可以通过以下步骤查看其平民的经验值等级：

- 进入“平民”选项卡以查看所有平民

RAID指挥官目标系统

Raid指挥官具有定位地块并自动为玩家发起Raid的独特能力。玩家不仅可以选择要自动Raid的地块，还可以通过在自动Raid目标图标界面中添加或删除收藏夹图标类型来轻松调整目标地块。

一个地块可以被Raid指挥官自动定位有两个先决条件：

- 目标地块必须至少从发起地块手动Raid过一次，并出现在已Raid地块的“最近列表”中

- 目标地块必须分配有一个收藏夹图标，并且该收藏夹图标必须添加到自动Raid目标图标界面中

大多数活跃的Raid玩家已经开发了自己的系统，使用现有的收藏夹图标来确定从其发起地块进行Raid的最佳和最差的房产。Raid Commander的自动目标系统允许玩家在新Auto-Raid目标界面上设置一个或多个收藏夹图标。通过在此界面上添加或删除收藏夹图标，玩家可以确定他们的Raid Commander应该专注于向哪些房产发起自动Raid。

每个房产都将有自己独特的Auto-Raid目标图标界面，使玩家能够自定义并为他们拥有的每个房产设置不同的Auto-Raid目标图标。例如，玩家可能专注于从一个房产Raid蓝色小鸟、绿色树木和红色樱桃，或者仅从另一个房产Raid蓝色小鸟。

更重要的是，如果玩家使用1级Raid Commander选择了蓝色小鸟和绿色树木图标，Raid Commander将从最近列表中找到最近的蓝色小鸟或绿色树木图标，并继续Raid和重新Raid该房产。如果玩家拥有2级Raid Commander，那么它将从最近列表中找到第二个最近的蓝色小鸟或绿色树木图标并Raid该房产，这意味着它将在两个不同的房产之间交替Raid，而不是一个，因为第二个最近的图标会不断更新，导致每次自动Raid发生时，一个新的房产都会占据其在最近列表中的位置。

这种动态目标系统将允许玩家快速有效地将他们的Raid Commander重新聚焦于不同的目标房产组。我们设计了这个系统，而不是一个数值分配的目标系统，因为后者会导致玩家需要手动设置静态列表，如果Raid成功率发生变化，则需要对单个房产进行精细的调整。

虽然我们独特的目标系统可能需要更长的时间才能理解，但我们相信它将更好地简化Auto-Raid功能，并且从长远来看，玩家需要进行的精细调整更少。

玩家将能够通过以下步骤使用Raid Commander设置Auto-Raid：

- 寻找“Auto Target Properties for Raid Commander”文本并按下随附的EDIT按钮。 这将可以通过现有的Raid界面和新的Civilian界面访问

![图片 8](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/raid-raid-commander-auto-raid-tab.jpg)

- 当选择“EDIT”时，玩家将看到Auto-Raid目标图标界面，允许他们设置Raid Commander针对所选地块要目标或忽略的偏好图标。

![图片 9](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/raid-auto-target-favorite-icon-selector.jpg)

- 在下面的这个特定案例中，玩家只选择了绿树作为目标图标。如果Raid Commander是1级，它将重复Raid MindeIslands。2级Raid Commander将Raid Suricato，然后是Mindelands。3级Raid Commander将Raid LOVEBTS，然后是Suricato，然后是Mindelands。在本例中，Raid Commander是6级，并且AUTO RAID按钮显示了Raid Commander将要攻击的地块顺序。此AUTO RAID列表由玩家选择的目标图标决定，因此玩家可以随时可视化他们的目标图标选择的结果。

![图片 10](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/image-7.png)

请注意，为了让Raid Commander自动Raid目标地块，该地块必须至少手动Raid过一次，应用了偏好图标，并且该偏好图标必须设置为目标图标。

合成CIVILIANS

首先，玩家将能够在4个 Tile 或更大的地块上合成3个Civilians。每个Civilian将需要25个E-ther和4个红色能量单位才能完全合成。每个Civilian每周需要持续4个红色能量单位来维持，比之前声明的1个红色能量单位有所增加。

一次只能合成一个Civilian，第一个Civilian是E-ther Reckoner (Dispenser)，第二个是Cydroid Technician (Recharger)，第三个可以合成的Civilian是Raid Commander。E-ther Reckoner还可以选择自动分配任何存储的E-ther，一旦它从Raid返回，或者等到存储已满后再分配。它也可以中止从5级分配Cydroid，以防万一将来可能发生突然情况需要该Cydroid。

要合成Civilian：

- 转到Civilian Management并选择Properties选项卡。

![图片 11](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/create-no-civilian-present.jpg)

- 选择地块，然后启动合成。最初，玩家可以选择每个地块合成1-3个Civilians。

![图片 12](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/image-3.png)

- Civilian可以一次合成一个，或者如果玩家确定要合成全部三个，则可以安排三个进行合成，一旦前一个Civilian完成合成，下一个Civilian将自动开始合成。合成的Civilian会按设定的顺序解锁一系列能力。因此，如果玩家需要在其地块上部署Raid Commander，他们需要首先合成前两个Civilian。

![图片 13](https://pub-60a5d261178e40e98b04d0c1a4bbcaea.r2.dev/news/image-1.png)

我们很高兴玩家能在不久的将来在我们的Stage2预生产服务器上探索新的Civilian功能。请密切关注我们的社交媒体，以获取有关Civilian正式发布的最新公告，但与以往一样，我们计划在正式发布公告之前，先在正式服务器上发布Civilian，并确保一切稳定。
